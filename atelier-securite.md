### Exercice 1 : Configuration d'une analyse de sécurité dans un pipeline Azure DevOps

**Objectif** : Mettre en place une analyse de sécurité automatisée dans un pipeline CI pour une application .NET Core.

**Étapes** :

1. Créer un nouveau pipeline Azure DevOps
2. Configurer l'intégration avec SonarCloud
3. Ajouter des analyses de vulnérabilité des dépendances
4. Définir des critères de qualité pour la validation du build

### Exercice 2 : Gestion des dépendances Open Source

**Objectif** : Mettre en place un processus de gestion des dépendances open source.

**Étapes** :

1. Configurer WhiteSource Bolt dans Azure DevOps
2. Analyser les dépendances d'un projet existant
3. Identifier et corriger les vulnérabilités détectées
4. Établir une politique de gestion des composants OSS

### Exercice 3 : Mesure et réduction de la dette technique

**Objectif** : Analyser la dette technique d'un projet et établir un plan de remboursement.

**Étapes** :

1. Configurer SonarCloud pour mesurer la dette technique
2. Analyser les résultats et identifier les problèmes prioritaires
3. Créer des tâches de remboursement dans Azure Boards
4. Intégrer la mesure de la dette technique dans les rapports d'état du projet
